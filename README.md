# Movies Frontend

Aplicación web que consume una API de películas y muestra un catálogo dinámico.

## Funcionalidades

- Visualización de películas
- Filtro por categoría
- Botón "Cargar más"
- Diseño responsivo básico

## Uso

Abrir el archivo `index.html` en el navegador.

## API

Este proyecto consume una API desarrollada en Laravel.
